# Asana App Builder Prototype

An interactive prototype of an AI-powered app builder built on top of Asana Tables.

**[→ Open the prototype](https://petertravis.github.io/asana-app-builder/asana-tables.html)**

## What it is

A single-file HTML prototype that lets you:
- Describe an app in plain English and watch it get built
- Chat with an AI assistant to modify the app (type `/` for a full command menu)
- Switch between Chat, Build, and Code views
- Preview the app in a realistic Asana UI chrome (mini rail, sidebar, tabs)
- Toggle mobile layout, dark mode, add views, blocks, and more
